# Data Analysis Project

This project focuses on analyzing parentalidade data to gain insights into parental support and policies. The dataset includes raw and processed data, and the analysis is performed using Jupyter notebooks.

## Project Structure

- `data/raw/parentalidade.csv`: Contains the raw dataset related to parentalidade.
- `data/processed/cleaned_data.csv`: Contains the cleaned version of the dataset after preprocessing.
- `notebooks/analysis.ipynb`: Jupyter notebook for performing data analysis and visualization.
- `src/preprocessing/clean_data.py`: Python script for cleaning and preprocessing the raw data.
- `src/visualization/plots.py`: Python script for creating visualizations of the data.
- `requirements.txt`: Lists the required Python packages for the project.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/MrGr33n98/datos-parentalidade.git
   ```

2. Navigate to the project directory:
   ```
   cd data-analysis-project
   ```

3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

- Use the `notebooks/analysis.ipynb` to perform data analysis and visualize results.
- Modify the scripts in the `src/preprocessing` and `src/visualization` directories as needed for custom analysis and visualizations.
